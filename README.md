# Telegram Username Finder Bot

Бот для поиска свободных юзернеймов в Telegram с системой Премиум, поддержкой и админ-панелью.

## Файлы

| Файл | Описание |
|------|----------|
| `bot.py` | Главный файл бота — вся логика хендлеров |
| `config.py` | Настройки: токен, ID админа, лимиты |
| `database.py` | Работа с SQLite базой данных |
| `username_checker.py` | Генерация и проверка юзернеймов |
| `requirements.txt` | Зависимости Python |

## Установка и запуск

```bash
pip install -r requirements.txt
python bot.py
```

База данных `bot_data.db` создаётся автоматически при первом запуске.

## Настройка (config.py)

| Параметр | Описание |
|----------|----------|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `ADMIN_ID` | Telegram ID владельца |
| `ADMIN_USERNAME` | Username владельца (без @) |
| `FREE_DAILY_LIMIT` | Лимит поисков для обычных пользователей |
| `PREMIUM_DAILY_LIMIT` | Лимит поисков для Премиум |
| `PREMIUM_PRICE` | Цена Премиума (для отображения) |
| `MAX_CHECK_ATTEMPTS` | Максимум попыток при поиске юзернейма |

## Функции

### Обычные пользователи
- 🔍 Поиск свободного юзернейма (по длине)
- 🎲 Случайный юзернейм
- ⭐ Профиль с лимитами
- 💎 Информация о Премиуме
- 📩 Написать в поддержку

### Премиум пользователи
- Всё то же, но 100 поисков в день вместо 10

### Админ панель (только для владельца)
- 📊 Статистика (пользователи, поиски, тикеты)
- 📢 Рассылка всем пользователям
- 💎 Выдача Премиума (/ban, /unban, через меню)
- 🚫 Бан / Разбан (/ban USER_ID, /unban USER_ID)
- 📨 Просмотр и ответ на тикеты поддержки
- 👑 Список Премиум пользователей
- 🗑 Удаление пользователя из базы (/deleteuser USER_ID)
- ✂️ Снятие Премиума (/revokepremium USER_ID)
